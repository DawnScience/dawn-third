The Usage Data Collector

The Usage Data Collector collects information about how individuals are using the DAWN platform. This information is periodically uploaded to servers hosted by Diamond Light Source. The intent is to use this data to help DAWN collaborators and developers better understand how DAWN is used.

Targeted Users of the Data:

  • Users of DAWN
  • Committers working on DAWN projects
  • ISVs and organizations creating DAWN based software
  • Enterprise IT departments that make extensive use of DAWN technology
